# 📅 简单日历 - 日记与作业记录应用

一个功能简洁但实用的安卓日历应用，专为学生和需要日常记录的用户设计。集成了日历视图、日记记录和作业管理功能。

## ✨ 功能特点

### 📆 日历功能
- **月视图日历**：清晰的月份展示，支持月份切换
- **日期选择**：点击日期可快速跳转到对应日期的记录
- **事件标记**：有记录的日期会显示橙色标记
- **今天高亮**：当前日期自动高亮显示

### 📝 日记记录
- **简单文本输入**：支持多行日记内容
- **日期关联**：日记自动与选定日期关联
- **数据持久化**：日记内容自动保存

### 📚 作业管理
- **8科目支持**：预设8个不同科目的作业记录
- **颜色区分**：每个科目使用不同颜色标识
- **科目自定义**：支持修改科目名称、提示文字和颜色
- **作业保存**：每日作业内容自动保存

### 🎨 界面设计
- **深色主题**：护眼友好的深色界面
- **Material Design**：遵循Material Design设计规范
- **响应式布局**：适配不同屏幕尺寸
- **直观操作**：简洁明了的用户交互

## 🚀 快速开始

### 系统要求
- Android 5.0 (API 21) 或更高版本
- 支持armeabi-v7a, arm64-v8a, x86, x86_64架构

### 安装方法
1. 下载最新版本的APK文件
2. 在安卓设备上启用"未知来源"安装权限
3. 找到并安装APK文件
4. 启动应用开始使用

## 📖 使用指南

### 基本操作
1. **查看日历**：应用启动后显示当前月份的日历
2. **切换月份**：点击顶部左右箭头按钮切换月份
3. **选择日期**：点击日历中的日期，自动加载该日期的记录

### 记录日记
1. 选择目标日期
2. 在底部日记区域输入内容
3. 点击"保存"按钮保存日记

### 记录作业
1. 选择目标日期
2. 在各个科目对应的输入框中填写作业内容
3. 点击"保存"按钮保存所有作业

### 自定义科目
1. 点击科目右侧的编辑按钮（铅笔图标）
2. 在弹出的对话框中修改科目名称、提示文字或颜色
3. 点击"保存"应用更改

## 🔧 技术特性

### 开发技术栈
- **编程语言**：Java
- **开发环境**：Android Studio
- **目标SDK**：API 34 (Android 14)
- **最低SDK**：API 21 (Android 5.0)

### 数据存储
- **本地存储**：使用SharedPreferences存储数据
- **数据格式**：JSON格式存储日记和作业内容
- **按日期组织**：每个日期的数据独立存储

### 架构设计
- **MVC架构**：清晰的数据、视图、控制分离
- **组件化**：模块化的功能设计
- **扩展性**：易于添加新功能

## 🛠️ 开发指南

### 环境搭建
1. 安装Android Studio Arctic Fox或更高版本
2. 配置Java 8或更高版本
3. 克隆或下载项目源码
